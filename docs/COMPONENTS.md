# Component Style Guide

## Button Components

### Primary Button
Used for main actions
- Class: btn-leaf-primary
- Gradient: leaf-500 to leaf-600

### Secondary Button
Used for secondary actions
- Class: btn-leaf-secondary
- Background: leaf-200

### Ghost Button
Used for tertiary actions
- Class: btn-leaf-ghost
- Background: transparent

## Card Components

### Default Card
Main content containers
- Class: card-leaf
- Border: leaf-400

### Elevated Card
Highlighted content
- Class: card-leaf-elevated
- Shadow: xl

### Glass Card
Overlay content
- Class: card-leaf-glass
- Blur: backdrop-blur-sm

## Game Components

### Game Grid
3x3 tic-tac-toe board
- Class: game-leaf-grid
- Gap: 3

### Game Cell
Individual board cells
- Class: game-leaf-cell
- Border: leaf-400

### Winner Cell
Winning combination cells
- Class: game-leaf-cell-winner
- Animation: glow-pulse

## Badge Components

### Status Badge
Shows game status
- Online: badge-leaf-online
- Waiting: badge-leaf-waiting

### Rank Badge
Shows leaderboard position
- Class: badge-leaf-rank
